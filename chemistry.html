2. Redox Reaction Shortcut

OIL RIG: Oxidation Is Loss (of electrons), Reduction Is Gain (of electrons)

3. Organic Chemistry

  • SN1 = Carbocation → rearrangement possible
  • SN2 = Backside attack → inversion of configuration
  • Markovnikov Rule = "Rich gets richer"

4. Thermodynamics Formulae

  • ΔG = ΔH – TΔS
  • Q = mcΔT
  • PV = nRT (Ideal Gas Law)
